# Snap

Browser extension for multi-resolution screenshots.

## Setup

```bash
npm install && npm run build
```

Load `dist` folder in `chrome://extensions` (Developer mode).

## Resolutions

| Format | Size | Use Case |
|--------|------|----------|
| Portrait | 1080×1350 | LinkedIn, Instagram feed |
| Landscape | 1200×630 | OG images, link previews |
| Vertical | 1080×1920 | Stories, Reels, Shorts |
| Square | 1080×1080 | Carousels, universal |
| Thumbnail | 400×300 | Previews |
